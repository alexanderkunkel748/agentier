---------------------------------------

2. Klicke auf deinen Payment Link
3. Gehe zu "After payment" / "Nach Zahlung"
4. Setze "Confirmation page" auf:
   https://agentier.netlify.app/promptinator.html?success=true

SCHRITT 2: Netlify Environment Variables setzen
------------------------------------------------
1. Gehe zu Netlify -> Deine Site -> Site Settings -> Environment Variables
2. Füge hinzu:
